# Sistema de Movimentação de Funcionários

## Objetivo

Automatizar a validação e movimentação de funcionários entre planilhas corporativas.

## Problema

Receber dois arquivos:

- TODOS.xlsx
- NOVOS.xlsx

Validar os registros utilizando CPF e Nome.

Gerar:

- Resultado.xlsx

## Regras de Negócio

- CPF como chave principal
- Validação de Nome
- Tratamento de duplicidades
- Tratamento de inconsistências
- Relatório de auditoria

## Tecnologias

- Excel
- Python
- Pandas
- OpenPyXL

## Resultados

- Redução de erros manuais
- Padronização dos dados
- Validação automática
- Relatórios de inconsistência
